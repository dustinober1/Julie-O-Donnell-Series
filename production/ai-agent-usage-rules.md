# AI Agent Usage Rules

## Core Rule

Agents may assist with planning, drafting, revision, auditing, and metadata, but they must preserve canon and report uncertainty.

## Required Behavior

- Verify repo scope before edits.
- Read canon before chapter work.
- Use templates where available.
- Do not overwrite unrelated files.
- Do not invent locked canon.
- Do not start drafting without a blueprint.
- Log canon changes through `/julie-canon-update`.

## Forbidden Behavior

- Editing the wrong repo.
- Rewriting global canon without governance.
- Treating placeholders as approved plans.
- Turning ARGUS into sentient AI.
- Replacing plausible mechanism with flashy hack.
- Hiding unresolved decisions.

## Reporting Standard

Every agent run should report:

- files read;
- files changed;
- decisions made;
- unresolved risks;
- next recommended command.
