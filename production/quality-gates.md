# Quality Gates

## Gate 1 — Repo Scope

No command edits files until repository scope is verified.

## Gate 2 — Canon Read

Every chapter command reads `.opencode/skills/julie-series-canon.md` and relevant book files.

## Gate 3 — Blueprint Before Draft

No chapter prose should be drafted without an approved chapter blueprint.

## Gate 4 — Continuity Pass

Every drafted chapter receives a continuity pass.

## Gate 5 — AI Plausibility

Every chapter involving AI, data, cyber, or systems must pass AI plausibility review.

## Gate 6 — Canon Update Governance

No global canon update without `/julie-canon-update` and a canon-change-log entry.

## Gate 7 — Publication Readiness

No export is publication-ready until full-book publication pass is complete.
