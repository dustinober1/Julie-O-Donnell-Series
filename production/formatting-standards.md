# Formatting Standards

## Chapter Files

Use Markdown.

```markdown
# Chapter X: Title

Chapter prose begins here.
```

## Reports

Use clear headings and severity labels.

## Naming

- Chapter files: `ch-001.md`, `ch-002.md`
- Prologue: `prologue.md`
- Revision reports: `ch-001-revision-report.md`
- Continuity reports: `ch-001-continuity-pass.md`
- Canon updates: `ch-001-canon-update.md`

## Placeholders

Placeholders must clearly state they are not draft prose.
