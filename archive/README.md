# Archive Policy

This directory records historical source provenance and archive rules.

Archived material is never active canon, never an active manuscript, and never a substitute for Git history.

## What belongs here

- descriptions of external migration sources;
- identifying hashes for historical snapshots not stored in the public repository;
- explicit notices that a source is superseded; and
- links to the active replacement.

## What does not belong here

- duplicate active manuscripts;
- extra files named `final`, `latest`, or `master`;
- unaccepted prose that is still under active review;
- copies of status files preserved only “just in case”; or
- private external document URLs that should not be published.

Superseded repository content is preserved through Git history. Active drafts belong in `books/book-01/drafts/`.
