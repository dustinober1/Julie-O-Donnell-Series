# /julie-draft-chapter

## Purpose

Draft a specific Julie O'Donnell chapter from an approved blueprint.

## Usage

```text
/julie-draft-chapter <book-number> <chapter-number>
```

## Required Skills

- `julie-series-canon`
- `julie-thriller-voice`
- `julie-ai-plausibility`
- `julie-prose-drafter`

## Task

1. Verify repo scope.
2. Read the current blueprint.
3. Read previous chapter prose if available.
4. Draft only the requested chapter.
5. Do not invent major plot, clue, or canon beyond the blueprint.
6. Preserve technical reveal pacing.
7. Save prose as manuscript-ready Markdown.

## Output Destination

```text
books/book-XX/chapters/ch-YYY.md
```

## Report

Include:

- chapter word count estimate;
- major scenes drafted;
- clue movement included;
- any blueprint gaps encountered;
- next recommended command.
