# /julie-continuity-pass

## Purpose

Audit a chapter for canon, timeline, clue, technical, and character continuity issues.

## Usage

```text
/julie-continuity-pass <book-number> <chapter-number>
```

## Required Skills

- `julie-series-canon`
- `julie-continuity-auditor`
- `julie-ai-plausibility`
- `julie-series-memory`

## Task

1. Verify repo scope.
2. Read the chapter, blueprint, clue map, timeline, continuity log, and chapter facts.
3. Check for conflicts with global series canon and book-specific canon.
4. Identify durable canon candidates but do not promote them.
5. Save a continuity report.

## Output Destination

```text
books/book-XX/continuity-passes/ch-YYY-continuity-pass.md
```

## Report Sections

- Files read
- Critical findings
- Recommended fixes
- Optional improvements
- Canon candidates
- Rejected trivia
- Next recommended command
