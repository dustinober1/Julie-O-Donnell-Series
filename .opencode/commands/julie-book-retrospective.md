# /julie-book-retrospective

## Purpose

Create a permanent post-book memory record after a Julie O'Donnell book is complete.

## Usage

```text
/julie-book-retrospective <book-number>
```

## Required Skills

- `julie-series-canon`
- `julie-series-memory`
- `julie-continuity-auditor`
- `julie-canon-update-governance`

## Task

1. Verify repo scope.
2. Read the final manuscript or final chapter set.
3. Read book-level QA, publication pass, and canon updates.
4. Summarize permanent consequences.
5. Identify what carries forward to future books.
6. Identify threat mechanics not to reuse too soon.
7. Mark unresolved threads.
8. Save the retrospective.

## Output Destination

```text
books/book-XX/book-retrospective.md
```

Also update when appropriate:

- `series-bible/10-book-by-book-series-arc.md`
- `series-bible/unresolved-series-threads.md`
- `series-bible/canon-change-log.md`

## Report

Include:

- book outcome;
- Julie arc movement;
- institutional consequences;
- recurring cast changes;
- technology/threat used;
- threads carried forward;
- next recommended series action.
