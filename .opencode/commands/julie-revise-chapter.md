# /julie-revise-chapter

## Purpose

Revise a drafted Julie O'Donnell chapter for commercial thriller quality while preserving story logic and canon.

## Usage

```text
/julie-revise-chapter <book-number> <chapter-number>
```

## Required Skills

- `julie-series-canon`
- `julie-thriller-voice`
- `julie-line-editor`
- `julie-ai-plausibility`

## Task

1. Verify repo scope.
2. Read the chapter, blueprint, and relevant book files.
3. Improve pacing, tension, dialogue, clarity, and scene turns.
4. Preserve all clues, canon, and technical mechanism.
5. Do not add major new plot without marking it as a risk.
6. Save the revised chapter in place only if the user requested edits.
7. Save a revision report.

## Output Destinations

Revised chapter:

```text
books/book-XX/chapters/ch-YYY.md
```

Revision report:

```text
books/book-XX/revision-notes/ch-YYY-revision-report.md
```

## Report

Include:

- major improvements;
- cuts or additions;
- continuity risks;
- next recommended command.
