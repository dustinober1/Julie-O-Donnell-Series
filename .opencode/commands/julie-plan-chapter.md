# /julie-plan-chapter

## Purpose

Create or repair a chapter blueprint for the requested Julie O'Donnell chapter.

## Usage

```text
/julie-plan-chapter <book-number> <chapter-number>
```

## Required Skills

- `julie-series-canon`
- `julie-chapter-architect`
- `julie-ai-plausibility`
- `julie-continuity-auditor`

## Task

1. Verify repo scope.
2. Read the chapter context packet or create one if missing.
3. Read book-level outline, clue map, mystery engine, and continuity log.
4. Produce a detailed blueprint using `templates/chapter-blueprint-template.md`.
5. Preserve clue timing and technical reveal ladder.
6. Mark unresolved issues instead of inventing major canon.

## Output Destination

```text
books/book-XX/chapter-blueprints/ch-YYY.md
```

## Report

Include:

- blueprint created or updated;
- key clue movement;
- canon risks;
- next recommended command.
