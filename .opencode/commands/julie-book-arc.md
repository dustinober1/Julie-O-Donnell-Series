# /julie-book-arc

## Purpose

Build or revise the full-book arc for a Julie O'Donnell installment without drafting chapter prose.

## Usage

```text
/julie-book-arc <book-number>
```

## Required Skills

- `julie-series-canon`
- `julie-series-memory`
- `julie-ai-plausibility`
- `julie-chapter-architect`

## Task

1. Verify repo scope.
2. Read the series bible and existing book files.
3. Define the book's promise, villain/motive, technical threat, clue ladder, Julie arc, institutional pressure, and ending shape.
4. Confirm the book uses a fresh AI/system failure mode and does not repeat prior books too closely.
5. Update or create:
   - `book-bible.md`
   - `ai-threat-design.md`
   - `mystery-engine.md`
   - `clue-map.md`
   - `red-herring-map.md`
   - `chapter-outline.md`
6. Keep unresolved items clearly marked as provisional.

## Output

Report:

- arc summary;
- files changed;
- unresolved decisions;
- canon candidates;
- next recommended command.
