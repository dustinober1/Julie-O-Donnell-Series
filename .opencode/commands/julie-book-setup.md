# /julie-book-setup

## Purpose

Create or repair the planning scaffold for a Julie O'Donnell book.

## Usage

```text
/julie-book-setup <book-number>
```

## Required Skills

- `julie-series-canon`
- `julie-series-memory`
- `julie-ai-plausibility`
- `julie-chapter-architect`

## Task

For the requested book:

1. Verify repository scope.
2. Read project and series canon files.
3. Check whether the book folder exists.
4. Create missing planning files using templates.
5. Do not draft prose.
6. Do not invent locked canon without recording it as provisional.
7. Update `progress.yaml` if appropriate.

## Expected Book Files

- `book-dashboard.md`
- `book-bible.md`
- `premise-and-hook.md`
- `target-reader.md`
- `antagonist-plan.md`

- `timeline.md`
- `character-arcs.md`
- `scene-list.md`
- `chapter-outline.md`
- `continuity-log.md`
- `chapter-facts.md`
- `revision-watchlist.md`
- `publication-checklist.md`
- `book-retrospective.md`

## Output

Report files created, files already present, and unresolved setup decisions.
