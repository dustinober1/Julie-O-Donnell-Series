# /julie-series-status

## Purpose

Produce a current status summary for the Julie O'Donnell series.

## Usage

```text
/julie-series-status
```

## Required Skills

- `julie-series-canon`
- `julie-series-memory`
- `julie-continuity-auditor`

## Read First

- `PROJECT_STATE.yaml`
- `progress.yaml`
- `series-bible/00-series-dashboard.md`
- `series-bible/10-book-by-book-series-arc.md`
- `series-bible/unresolved-series-threads.md`
- any available book dashboard files under `books/`

## Task

Summarize:

- active book status;
- production hold status;
- completed scaffolding;
- missing core files;
- unresolved canon questions;
- next safe production step.

## Output

Use headings:

1. Repository Scope
2. Active Book
3. Series Canon Status
4. Production Readiness
5. Open Risks
6. Recommended Next Command
