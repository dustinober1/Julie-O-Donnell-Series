# /julie-ai-tell-sweep

## Purpose

Remove or report prose patterns that make a Julie O'Donnell chapter feel generated, repetitive, generic, or emotionally over-labeled.

## Usage

```text
/julie-ai-tell-sweep <book-number> <chapter-number>
```

## Required Skills

- `julie-thriller-voice`
- `julie-line-editor`
- `julie-ai-tell-sweeper`

## Task

1. Verify repo scope.
2. Read the chapter and revision report if present.
3. Find repeated phrases, repeated gestures, generic emotional labels, and over-polished dialogue.
4. Edit only for style unless the user requests deeper revision.
5. Preserve clue logic and canon.
6. Save report and updated chapter when edits are made.

## Output

Save sweep report to:

```text
books/book-XX/revision-notes/ch-YYY-ai-tell-sweep.md
```

Report:

- repeated phrases found;
- edits made;
- phrases intentionally preserved;
- remaining risks;
- next recommended command.
