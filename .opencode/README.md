# OpenCode Workflow

This folder contains the Julie O'Donnell series-specific OpenCode skills and slash commands.

## Canonical Skills Path

Skills live in:

```text
.opencode/skills/
```

The master canon skill is:

```text
.opencode/skills/julie-series-canon.md
```

## Slash Commands Path

Commands live in:

```text
.opencode/commands/
```

## Core Rule

Chapter commands may discover canon. Only canon-maintenance commands may promote canon.

## First Commands to Use After Review

```text
/julie-series-status
/julie-chapter-context 1 prologue
/julie-plan-chapter 1 prologue
```
