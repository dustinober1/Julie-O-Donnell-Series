# Julie O’Donnell Series Controls

This directory stores continuity that must survive beyond one book.

Book-specific accepted prose and canon controls remain inside each book directory. Series controls may summarize accepted facts and intended recurrence, but they cannot overrule a book’s accepted manuscript.

## Current series control

- [`recurring-character-ledger.md`](recurring-character-ledger.md)

## Update rule

Update the series ledger whenever accepted prose changes a recurring character’s:

- name, rank, title, or affiliation;
- physical condition or permanent mark;
- public, legal, employment, or clearance status;
- location;
- knowledge or false belief;
- evidence, secret, or leverage;
- relationship with Julie;
- Book 1 end state;
- return trigger;
- future-book function; or
- no-retcon lock.

Planning a return is not the same as canonically guaranteeing survival, employment, allegiance, or availability.
