# Book 1 Manuscript Folder

This folder will hold drafted chapters for Book 1 of the Julie O'Donnell Series.

## Naming convention

Use:

```text
ch-001.md
ch-002.md
ch-003.md
...
```

Use `prologue.md` only if the final project chooses to keep the prologue outside chapter numbering. Otherwise, use `ch-000-prologue.md` or `ch-001.md` consistently and document the choice in `books/book-01/outline.md`.

## Before drafting

Do not add chapter prose until the relevant chapter blueprint exists and the control docs have been loaded.

Required reads:

- `PROJECT_STATE.yaml`
- `docs/03_Book_1_Story_Bible.md`
- `docs/04_Character_Bible.md`
- `docs/05_Technology_and_Threat_Model.md`
- `docs/06_Continuity_Canon_Locks.md`
- `docs/07_Prose_Style_and_Voice_DNA.md`
- `docs/08_Chapter_Production_Pipeline.md`
- `books/book-01/outline.md`

## Chapter acceptance checklist

A chapter must pass:

1. Story function check.
2. Thriller pressure check.
3. Julie competence check.
4. ARGUS/Apex continuity check.
5. Anti-AI prose cleanup.
6. Chapter-ending hook check.
7. Next-chapter handoff.
